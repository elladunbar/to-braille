# to-braille

This project is meant to help display images in terminals and terminal emulators
without support for sophisticated graphics display protocols like sixel. It
requires a terminal with font support for braille unicode characters.

Images will be displayed in a two-tone style with optional dithering support.
The image will keep its aspect ratio and be displayed with the maximum height
and width available to it.

Following popular trends, the program assumes you are using a terminal in dark
mode.

## Installation

1. Clone this repository, and enter the newly created directory.
2. Install with `cargo install --path .`

## Usage

```
Usage: to-braille [OPTIONS] <IMAGE>

Arguments:
  <IMAGE>
          Path to image

Options:
  -d, --dithering <DITHERING>
          Choose dithering type

          Possible values:
          - bayer: Bayer matrix (size 4)
          - blue:  Blue noise
          - white: White noise

  -h, --help
          Print help (see a summary with '-h')

  -V, --version
          Print version
```
